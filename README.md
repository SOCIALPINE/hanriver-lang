# 코딩
꽁,고,양,이,얼어붙은,돌 6개 키워드로 이루어진 언어입니다.
+주의:모든 줄 마지막에는 ; 을 붙여야 합니다.
#+,-,*,/
꽁: +1
고: -1
양: *
이: /
# 돌:변수설정
예:
돌꽁꽁;
첫번째 변수에 2 저장
돌돌고고고;
두번째 변수에 -3 저장
# 얼어붙은~:출력하기
예:
돌꽁꽁;
얼어붙은돌~;
출력:2
얼어붙은꽁꽁고~;
출력:1
주의:얼어붙은 으로 시작하고 ~ 으로 끝나야 합니다
